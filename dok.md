# ДЗ урок №2 Войтенко Юрий

Создание репозитория:
```sh
git init
```

Добавить отдельный файл:
```sh
git add name.file
```

Добавить все файлы:
```sh
git add .
```

Добавление коммита:
```sh
git commit -m 'Message'
```

Просмотр изменений:
```sh
git log
```

Перемещение на предыдущие сохраниения:
```sh
git checkout ...
```
